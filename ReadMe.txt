O site conta com 4 páginas, sendo elas: 
  - A página inicial que conta com alguns produtos em destaque eu um carrossel de fotos
  - A página de login/registrar-se, está possui dois formulários, uma para se registrar e outro para fazer login
  - A página de Banho e Tosa, onde o cliente poderá registar seu pet e agendar o serviço com o petshop
  - A página de produtos, onde é possível ver todos os produtos e filtra-lós pela sua preferência

Todas as páginas possuem um rodapé e um cabeçalho onde é possível ver as informações do petshop e navegar pelo site respectivamente

O site se encontra dividido em pastas, sendo pages para os arquivos html e styles para seus respectivos arquivos CSS
